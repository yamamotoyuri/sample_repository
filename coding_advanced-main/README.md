1.右上のcodeからDowload zipでフォルダをダウンロードしてください。

2.ダウンロードしたフォルダを開いて中身に

  - img
  - 見本
    - PC.png
    - SP.png
    - PC動作確認.MOV
    - SP動作確認.MOV

3.中身が正しいことを確認したらレスポンシブサイト制作.pdfの内容に沿ってカリキュラムを進めてください。
